python-生成工程依赖包文件

2020-02-23

在使用python进行编程时，往往会用到很多依赖包，pipreqs工具能够帮助我们扫描某一工程，并生成其所使用到的所有依赖包清单。
首先确认计算机中是否安装有pipreqs工具，如果没有，安装此工具。
具体做法是，打开terminal，键入指令pip3 install pipreqs，安装完成后确认是否安装成功（键入指令pipreqs --version如果安装没有报错可以不做），安装完成后进入你要查询的工程目录（pycharm中的终端操作也可以），键入指令pipreqs ./，完成后会生成名字为requirements.txt的文本文档，里面就是该工程下所有依赖包及其版本。
